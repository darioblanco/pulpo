# CLI Reference

Top-level commands:

```text
pulpo spawn --workdir <PATH> [PROMPT...]
pulpo list
pulpo logs <NAME> [--follow]
pulpo attach <NAME>
pulpo input <NAME> [TEXT]
pulpo kill <NAME>
pulpo delete <NAME>
pulpo resume <NAME>
pulpo interventions <NAME>
pulpo nodes
pulpo schedule <install|list|pause|resume|remove>
pulpo ui
```

For exact options, run:

```bash
pulpo --help
pulpo <command> --help
```
