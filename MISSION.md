# Mission

Pulpo is a self-hosted control plane for running coding agents reliably across your own machines.

It provides:
- a durable runtime for agent sessions,
- policy and safety guardrails,
- observable lifecycle events,
- and interface-agnostic control via API, CLI, and web UI.

Pulpo is infrastructure, not a prompt framework.

## Non-Goals

- Defining the "best" personas or prompting methodology
- Replacing specialized local agent UX tools
- Building a monolithic all-in-one platform
